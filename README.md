# microscope-meteor
First example meteor app

### Version
0.0.1

### Installation

You only need install meteor

```sh
$ curl https://install.meteor.com/ | sh
```
 and run
 ```sh
$ meteor
 ```

# Dependencies
Microscope needs to add some Meteor Packages:
  - garciagon:errors 
> To manage errors on app
```sh
$ meteor add garciagon:errors
```
